# To-Do App

A clean, interactive to-do list application built with JavaScript, HTML, and CSS. Features drag-and-drop reordering, local storage persistence, dark mode, and a celebration modal when all tasks are completed.

## Features

- **Add Tasks** - Quickly add new to-do items
- **Mark Complete** - Single click to toggle completion status
- **Delete Tasks** - Double click to delete individual tasks
- **Drag & Drop** - Reorder tasks by dragging them to your preferred position
- **Local Storage** - All tasks are automatically saved and persist between sessions
- **Dark Mode** - Toggle between light and dark themes with preference saved

- **Delete All** - Clear your entire list with one click

## How to Use

### Adding a Task
1. Type your task in the input field
2. Press Enter or submit the form

### Marking a Task Complete
- **Single click** on any task to mark it as complete (or unmark if already complete)
- Completed tasks are shown with a strikethrough

### Deleting a Task
- **Double click** quickly (within 240ms) on a task to delete it
- The timing window is intentionally tight to prevent accidental deletions

### Reordering Tasks
- Click and drag any task to move it to a new position in the list

### Dark Mode
- Click the "Dark Mode" button to toggle between light and dark themes
- Your preference is saved and will persist on your next visit

### Clearing All Tasks
- Click the "Delete List" button to remove all tasks at once

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Blotnoir/todo-app.git
   ```

2. Navigate to the project directory:
   ```bash
   cd todo-app
   ```

3. Open `index.html` in your web browser

That's it! No build process or dependencies required.
